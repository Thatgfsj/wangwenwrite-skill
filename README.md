✨ 项目介绍

wangwenwrite-skill是一款专为男频系统流网文创作者打造的全流程辅助技能，聚焦2024-2026年行业核心趋势，覆盖创作全链路，旨在降低创作门槛、优化内容质量、提升作品竞争力，适配新手、资深作者及应急创作等多种场景。

核心定位：不替代作者创意，只做高效创作助手，专注解决系统流网文“设定崩坏、爽点失衡、AI味儿重、趋势脱节”四大痛点。

📌 核心技能（Skills）

- 选题趋势研判：贴合2024-2026年男频系统流爆款方向，提供选题建议

- 核心设定搭建：系统机制、人设反差、世界观细化，支持情绪值/躺平变强等新型系统

- 分层大纲撰写：开篇3章详纲、中期节点、后期反转，遵循“300字钩子+3章觉醒”节奏

- 章节内容创作：黄金七行钩子、爽点密度调控、情绪过山车设计，规避AI味儿

- 文本润色优化：逻辑校验、AI化表述修改、爽点强化，标注趋势适配点

- 发布适配调整：标题建议、章节钩子提炼、多平台调性适配（起点/番茄/飞卢）

- 创作辅助补充：需求解析、瓶颈突破、反转剧情生成、应急内容产出

🔧 功能特性

1. 趋势适配：深度贴合2025-2026年核心趋势（情绪价值优先、类型融合、反常识金手指）

2. 全流程覆盖：从选题到发布6大核心环节，支持单一环节精准调用或全流程联动

3. 低AI味儿：内置专属检查清单，替换模板化套话，补充细节与具体情绪表达

4. 高兼容性：适配短篇（10-20章）、长篇（百章+），兼容主流男频网文平台

5. 易上手：提供系统设定、大纲、章节等标准化模板，新手可直接套用

🎯 适用场景

- 新手创作者：从零搭建系统流网文框架，快速掌握爆款创作逻辑

📋 执行流程（SOP）

1. 需求解析：提取用户需求、匹配行业趋势，输出解析报告与执行重点

2. 核心设定：搭建系统、人设、世界观，明确能力边界与趋势适配点

3. 大纲撰写：输出分层大纲，标注钩子位置、名场面与反转方向

4. 章节创作：按节奏生成章节内容，融入钩子、爽点与系统交互

5. 润色优化：修改AI化表述、校验逻辑，补充优化说明

6. 发布适配：提供标题建议、章节钩子与日更节奏指导
